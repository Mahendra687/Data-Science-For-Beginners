**เดเตเดฑเดฟเดชเตเดชเต**: เดจเดพเด เดฎเตเดฎเตเดชเดคเตเดคเต เดชเดเตเดเดฟเด เดเตเดฐเดพเตปเดธเตเดชเตเดธเต เดเตเดฏเตเดคเต (เดเดฅเดตเดพ `.T` เดเดชเดฏเตเดเดฟเดเตเดเต) เด เดชเดเตเดเดฟเด เดฐเตเดชเด เดฒเดญเดฟเดเตเดเดพเดฎเตเดจเตเดจเต เดถเตเดฐเดฆเตเดงเดฟเดเตเดเตเด, เดเดฆเดพ.

เดเดตเดฟเดเต `.T` เดกเดพเดฑเตเดฑเดพเดซเตเดฐเตเดฏเดฟเดฎเดฟเดจเตเดฑเต เดเตเดฐเดพเตปเดธเตเดชเตเดธเดฟเดเดเต (เดตเดฐเดฟ, เดเตเดณเดเตเดเตพ เดฎเดพเดฑเตเดฑเตฝ) เดชเตเดฐเดตเตผเดคเตเดคเดจเดฎเดพเดฃเต, `rename` เดชเตเดฐเดตเตผเดคเตเดคเดจเด เดเตเดณเด เดชเตเดฐเตเดเตพ เดฎเตเตปเดชเดคเตเดคเต เดเดฆเดพเดนเดฐเดฃเดคเตเดคเดฟเดจเต เดเดจเตเดฏเตเดเตเดฏเดฎเดพเดฏเดฟ เดฎเดพเดฑเตเดฑเดพเตป เดธเดนเดพเดฏเดฟเดเตเดเตเดจเตเดจเต.

เดกเดพเดฑเตเดฑเดพเดซเตเดฐเตเดฏเดฟเดฎเตเดเดณเดฟเตฝ เดจเดพเด เดเตเดฏเตเดฏเดพเดตเตเดจเตเดจ เดเดฟเดฒ เดชเตเดฐเดงเดพเดจ เดชเตเดฐเดตเตผเดคเตเดคเดจเดเตเดเตพ:

**เดเตเดณเด เดคเดฟเดฐเดเตเดเตเดเตเดชเตเดชเต**. `df['A']` เดเดดเตเดคเดฟเดฏเดพเตฝ เดตเตเดฏเดเตเดคเดฟเดเดค เดเตเดณเด เดคเดฟเดฐเดเตเดเตเดเตเดเตเดเดพเด - เดเดคเต เดเดฐเต เดธเตเดฐเตเดธเต เดจเตฝเดเตเด. `df[['B','A']]` เดเดดเตเดคเดฟเดฏเดพเตฝ เดเตเดณเดเตเดเดณเตเดเต เดเดฐเต เดเดชเดธเดฎเตเดนเด เดฎเดฑเตเดฑเตเดฐเต เดกเดพเดฑเตเดฑเดพเดซเตเดฐเตเดฏเดฟเดฎเดฟเตฝ เดคเดฟเดฐเดเตเดเตเดเตเดเตเดเดพเด.

**เดจเดฟเดฌเดจเตเดงเดจเดเตพ เดชเตเดฐเดเดพเดฐเด** เดเดฟเดฒ เดตเดฐเดฟเดเตพ เดฎเดพเดคเตเดฐเด เดซเดฟเตฝเดเตเดเตผ เดเตเดฏเตเดฏเตฝ. เดเดฆเดพเดนเดฐเดฃเดคเตเดคเดฟเดจเต, `A` เดเตเดณเดคเตเดคเดฟเดฒเต เดฎเตเดฒเตเดฏเด 5-เตฝ เดเตเดเตเดคเดฒเตเดณเตเดณ เดตเดฐเดฟเดเตพ เดฎเดพเดคเตเดฐเด เดตเตเดฃเดฎเตเดเตเดเดฟเตฝ, `df[df['A']>5]` เดเดดเตเดคเดพเด.

> **เดเตเดฑเดฟเดชเตเดชเต**: เดซเดฟเตฝเดเตเดเดฑเดฟเดเดเต เดชเตเดฐเดตเตผเดคเตเดคเดฟเดเตเดเตเดจเตเดจเดคเต เดเดเตเดเดจเต เดเดฃเต. `df['A']<5` เดเดจเตเดจ เดชเตเดฐเดเดเดจเด เดเดฐเต เดฌเตเดณเดฟเดฏเตป เดธเตเดฐเตเดธเต เดจเตฝเดเตเดจเตเดจเต, เดเดคเต `df['A']` เดฏเดฟเดฒเต เดเดฐเต เดฎเตเดฒเตเดฏเดคเตเดคเดฟเดจเตเด `True` เดเดฒเตเดฒเตเดเตเดเดฟเตฝ `False` เดเดฃเต. เดฌเตเดณเดฟเดฏเตป เดธเตเดฐเตเดธเต เดเตปเดกเดเตเดธเต เดเดฏเดฟ เดเดชเดฏเตเดเดฟเดเตเดเตเดฎเตเดชเตเตพ, เดกเดพเดฑเตเดฑเดพเดซเตเดฐเตเดฏเดฟเดฎเดฟเดฒเต เดตเดฐเดฟเดเดณเตเดเต เดเดชเดธเดฎเตเดนเด เดคเดฟเดฐเดฟเดเต เดจเตฝเดเตเดจเตเดจเต. เดเดคเดฟเดจเดพเตฝ, เดธเดพเดงเดพเดฐเดฃ Python เดฌเตเดณเดฟเดฏเตป เดชเตเดฐเดเดเดจเด เดเดชเดฏเตเดเดฟเดเตเดเดพเดจเดพเดเดฟเดฒเตเดฒ, เดเดฆเดพ., `df[df['A']>5 and df['A']<7]` เดคเตเดฑเตเดฑเดพเดฃเต. เดชเดเดฐเด, เดฌเตเดณเดฟเดฏเตป เดธเตเดฐเตเดธเตเดเดณเดฟเตฝ เดชเตเดฐเดคเตเดฏเตเด `&` เดเดชเตเดชเดฑเตเดทเตป เดเดชเดฏเตเดเดฟเดเตเดเต, `df[(df['A']>5) & (df['A']<7)]` เดเดดเตเดคเดฃเด (*เดฌเตเดฐเดพเดเตเดเดฑเตเดฑเตเดเตพ เดจเดฟเตผเดฃเดพเดฏเดเดฎเดพเดฃเต*).

**เดชเตเดคเดฟเดฏ เดเดฃเดเตเดเดพเดเตเดเดพเดตเตเดจเตเดจ เดเตเดณเดเตเดเตพ เดธเตเดทเตเดเดฟเดเตเดเตฝ**. เดจเดพเด เดเดเตเดเดจเต เดฒเดณเดฟเดคเดฎเดพเดฏ เดชเตเดฐเดเดเดจเด เดเดชเดฏเตเดเดฟเดเตเดเต เดกเดพเดฑเตเดฑเดพเดซเตเดฐเตเดฏเดฟเดฎเดฟเดจเต เดชเตเดคเดฟเดฏ เดเดฃเดเตเดเดพเดเตเดเดพเดตเตเดจเตเดจ เดเตเดณเดเตเดเตพ เดธเตเดทเตเดเดฟเดเตเดเดพเด:
```python

```
